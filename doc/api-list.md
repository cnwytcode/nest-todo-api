API LIST

## User API

```sh
GET    /api/v1/user/     show list (findAll)
GET    /api/v1/user/:id  show detail (findOne)
POST   /api/v1/user/     add item (add item)
PUT    /api/v1/user/:id  edit item (edit item)
DELETE /api/v1/user/:id  delete item (delete item)
```
